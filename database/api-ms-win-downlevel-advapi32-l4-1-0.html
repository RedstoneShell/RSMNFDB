Description: This is a DOWNLEVEL API Set forwarder DLL for system shutdown control, account lookup, and LSA (Local Security Authority) operations. It contains NO executable code — only export forwarders to advapi32.dll. These functions manage system shutdown, account name resolution, and privilege lookups.
