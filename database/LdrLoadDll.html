LdrLoadDll

UNDOCUMENTED WIN 7 / NT 6.1 ntdll.dll

Description:Low level analog of "LoadLibrary" function, load DLL in process address space. Used by all DLL loaders. Returns a NTSTATUS.

C# P/Invoke Structure:

[DllImport("ntdll.dll", SetLastError = true, CharSet = CharSet.Unicode)]
public static extern int LdrLoadDll(
    string DllPath, 
    ref uint DllCharacteristics, 
    ref UNICODE_STRING DllName, 
    out IntPtr DllHandle
);

Required Structure (UNICODE_STRING):

[StructLayout(LayoutKind.Sequential, CharSet = CharSet.Unicode)]
public struct UNICODE_STRING {
    public ushort Length;
    public ushort MaximumLength;
    [MarshalAs(UnmanagedType.LPWStr)]
    public string Buffer;

    public UNICODE_STRING(string s) {
        Length = (ushort)(s.Length * 2);
        MaximumLength = (ushort)(Length + 2);
        Buffer = s;
    }
}

Possible NTSTATUS Codes:

0x00000000 - STATUS_SUCCESS - Operation completed successfully
[0x00000000]

0xC0000002 - STATUS_NOT_IMPLEMENTED - Function not implemented
[0xC0000002]

0xC0000022 - STATUS_ACCESS_DENIED - Insufficient privileges
[0xC0000022]

0xC0000005 - STATUS_ACCESS_VIOLATION - Invalid memory access
[0xC0000005]

0xC0000034 - STATUS_OBJECT_NAME_NOT_FOUND - DLL not found
[0xC0000034]

        0xC0000008 - STATUS_INVALID_HANDLE - Invalid handle passed
[0xC0000008]

0xC0000142 - STATUS_DLL_INIT_FAILED - DLL entry point initialization failed
[0xC0000142]

0xC000012F - STATUS_INVALID_IMAGE_FORMAT - Not a valid DLL
[0xC000012F]

Example Usage:

UNICODE_STRING us = new UNICODE_STRING("kernel32.dll");
uint flags = 0;
IntPtr hModule;

int status = LdrLoadDll(null, ref flags, ref us, out hModule);

if(status == 0) // STATUS_SUCCESS
{
    Console.WriteLine("DLL loaded successfully: " + hModule);
}
else
{
    Console.WriteLine("Failed to load DLL. NTSTATUS: 0x" + status.ToString("X8"));
}

⚠️ WARNING:

Parameter DllPath can be NULL, in this case Windows uses standard DLL search. Function may throw AccessViolationException if UNICODE_STRING.Buffer is not initialized correctly!
