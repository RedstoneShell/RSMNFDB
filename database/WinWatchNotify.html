WinWatchNotify

UNDOCUMENTED WIN 7 / NT 6.1+ dciman32.dll

Description: Notifies WinWatch objects of status changes. This function is exported but appears to be a stub in Windows 7. It may be called internally by DCIBeginAccess when a surface is lost.

C# P/Invoke Structure:

[DllImport("dciman32.dll", SetLastError = true)]
public static extern void WinWatchNotify();

⚠️ WARNING:

This function may not be implemented. Status changes are handled automatically by DCIBeginAccess.
