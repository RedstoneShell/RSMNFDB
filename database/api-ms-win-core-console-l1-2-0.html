api-ms-win-core-console-l1-2-0.dll

API SET FORWARDER KERNEL32.DLL Windows 8+ / NT 6.2+ 17 exports

Description: This is an API Set forwarder DLL for console management functions in Windows. It contains NO executable code — only export forwarders to kernel32.dll. These functions provide comprehensive control over console windows, including creation, input/output operations, mode configuration (VT100/ANSI support), and console event handling.

Why it exists: This is the extended console API set introduced in Windows 8. Compared to l1-1-0, it adds AttachConsole (attach to another process's console) and FreeConsole (detach from console), as well as Unicode versions of input peek functions. These functions enable modern terminal emulators and tools like Windows Terminal, ConEmu, and Cmder.

🔗 Forwarder Example (C):

// This DLL contains NO actual implementation
// All console functions are forwarded to kernel32.dll

// Console creation forwarders
#pragma comment(linker, "/export:AllocConsole=kernel32.AllocConsole")
#pragma comment(linker, "/export:AttachConsole=kernel32.AttachConsole")

// Console mode forwarders (VT100/ANSI support)

⚠️ WARNING:

Modern console features require Windows 8+ or Windows 10!
• AttachConsole and FreeConsole are Windows 8+ only
• VT100/ANSI escape sequences (colors, cursor movement) require SetConsoleMode with ENABLE_VIRTUAL_TERMINAL_PROCESSING (Windows 10)
• True color (24-bit) support requires Windows 10 Fall Creators Update (1709)
• Pseudoconsole (ConPTY) functions are in a separate API Set (l2-1-0)
• For Windows 7, console functions have limited VT100 support (no true color)
• For ReactOS: implement AttachConsole and modern console modes for terminal emulator compatibility
