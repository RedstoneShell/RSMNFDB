api-ms-win-core-file-l1-2-0.dll

API SET FORWARDER KERNEL32.DLL Windows 8 / NT 6.2+ 4 exports

Description: This is an API Set forwarder DLL for core file operations in Windows. It contains NO executable code — only export forwarders to kernel32.dll. These functions provide extended file operations introduced in Windows 8, including enhanced file creation and volume management APIs.

Why it exists: This API Set provides the Windows 8+ file API extensions. The key function is CreateFile2, which extends the classic CreateFile with additional parameters for modern Windows features. Also includes volume management functions that work with mount points and volume names. This API Set helps applications maintain compatibility across Windows versions.

📦 Exported Functions

Function Forwarded To Description
CreateFile2 kernel32.CreateFile2 Creates or opens a file with extended parameters (Windows 8+)
GetTempPathW kernel32.GetTempPathW Retrieves the path of the temporary directory
GetVolumeNameForVolumeMountPointW kernel32.GetVolumeNameForVolumeMountPointW Gets a volume GUID path for the volume associated with a mount point
GetVolumePathNamesForVolumeNameW kernel32.GetVolumePathNamesForVolumeNameW Retrieves a list of drive letters and mount points for a specified volume

⚠️ WARNING:

Windows 8+ only!
• This API Set is not available on Windows 7 or earlier
• CreateFile2 requires Windows 8 or Windows Server 2012 minimum
• Volume management functions work with volume GUID paths (\\?\Volume{...})
• For Windows 7 compatibility, use classic CreateFile instead
• These functions are part of the Core File API family (l1-2-0 = Level 1, version 2.0)
• For ReactOS: implement these functions for Windows 8+ API compatibility
