RtlGetVersion(RTL_OSVERSIONINFOW lpVersionInformation)

PATRIALY DOCUMENTED WIN 7 / NT 6.1 ntdll.dll

Опис:Return as `lpVersionInformation` version of installed OS, firset documented func at RSMNFDB.

C# P/Invoke Structure:

[DllImport("ntdll.dll", SetLastError = true)]
public static extern int RtlGetVersion(ref RTL_OSVERSIONINFOW lpVersionInformation);

Associated Structure:

[StructLayout(LayoutKind.Sequential, CharSet = CharSet.Unicode)]
public struct RTL_OSVERSIONINFOW {
    public uint dwOSVersionInfoSize;
    public uint dwMajorVersion;
    public uint dwMinorVersion;
    public uint dwBuildNumber;
    public uint dwPlatformId;
    [MarshalAs(UnmanagedType.ByValTStr, SizeConst = 128)]
    public string szCSDVersion;
}

Field Explanation:

  • dwOSVersionInfoSize: Size of this structure in bytes.
  • dwMajorVersion: Major version number of the OS.
  • dwMinorVersion: Minor version number of the OS.
  • dwBuildNumber: OS build number.
  • dwPlatformId: Platform ID (e.g., NT, Windows 9x).
  • szCSDVersion: Service pack string (Unicode, up to 128 chars).

Example Usage:

RTL_OSVERSIONINFOW osInfo = new RTL_OSVERSIONINFOW();
osInfo.dwOSVersionInfoSize = (uint)Marshal.SizeOf(typeof(RTL_OSVERSIONINFOW));
uint status = RtlGetVersion(ref osInfo); // P/Invoke to ntdll.dll
if (status == 0x00000000) {
    Console.WriteLine($"OS Version: {osInfo.dwMajorVersion}.{osInfo.dwMinorVersion} Build {osInfo.dwBuildNumber}");
}
