NtCreateThreadEx

UNDOCUMENTED WIN 7 / NT 6.1 ntdll.dll

Description:Creates a thread in a specified process. This function is used to create fully-controllable threads using PS_ATTRIBUTE_LIST (e.g., selecting a specific CPU core or hiding the thread from debuggers).

C# P/Invoke Structure:

[DllImport("ntdll.dll", SetLastError = true)]
public static extern uint NtCreateThreadEx(
    out IntPtr threadHandle,
    uint desiredAccess,
    IntPtr objectAttributes,
    IntPtr processHandle,
    IntPtr startRoutine,
    IntPtr argument,
    uint createFlags,
    UIntPtr zeroBits,
    UIntPtr stackSize,
    UIntPtr maximumStackSize,
    IntPtr attributeList
);

Creation Flags (Examples):

        THREAD_CREATE_FLAGS_CREATE_SUSPENDED = 0x00000001;
        THREAD_CREATE_FLAGS_SKIP_THREAD_ATTACH = 0x00000002; // No DLL_THREAD_ATTACH
        THREAD_CREATE_FLAGS_HIDE_FROM_DEBUGGER = 0x00000004;

⚠️ WARNING:

Using this function for code injection into system processes requires SeDebugPrivilege. Incorrectly filling the PS_ATTRIBUTE_LIST will immediately crash the calling process.
