DOWNLEVEL API SET SECURITY & REGISTRY ADVAPI32.DLL Windows 7+ (with KB2999226) 160 exports

Description: This is a DOWNLEVEL API Set forwarder DLL for security, registry, and event tracing functions. It contains NO executable code — only export forwarders to advapi32.dll. These functions provide comprehensive Windows security infrastructure: access control (ACL), privilege management, SID manipulation, registry operations, and Event Tracing for Windows (ETW).

Why it exists: This API Set provides backward compatibility for security-related APIs introduced in Windows 8/10. Modern applications that use advanced security features (like mandatory integrity control, fine-grained access checks, or ETW) can run on Windows 7 through this forwarder layer.
