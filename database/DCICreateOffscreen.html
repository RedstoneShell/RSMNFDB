DCICreateOffscreen

UNDOCUMENTED WIN 7 / NT 6.1+ dciman32.dll

Description: Creates an offscreen DCI surface (not visible on display). This function is exported but its implementation appears to be a stub or may be handled by DirectDraw internally. Typically used for offscreen rendering buffers.

C# P/Invoke Structure:

[DllImport("dciman32.dll", SetLastError = true)]
public static extern int DCICreateOffscreen(IntPtr hdc, uint dwWidth, uint dwHeight, uint dwFormat, out IntPtr ppSurfaceInfo);

Parameters:

hdc         [in] - HDC handle from DCIOpenProvider
dwWidth     [in] - Width of the surface in pixels
dwHeight    [in] - Height of the surface in pixels
dwFormat    [in] - Pixel format (0x200 = RGB, etc.)
ppSurfaceInfo [out] - Receives pointer to DCISURFACE structure

Return Codes:

0x00000000 - STATUS_SUCCESS - Surface created successfully
0xC0000002 - STATUS_NOT_IMPLEMENTED - Stub implementation (Wine)
    [0xC0000002]

⚠️ WARNING:

This function may be deprecated. Use DirectDraw APIs for offscreen surfaces instead.
