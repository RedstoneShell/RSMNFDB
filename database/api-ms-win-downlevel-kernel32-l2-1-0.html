⚠️ WARNING:

Legacy APIs — use with caution!
• GlobalAlloc/LocalAlloc are from 16-bit Windows era — use HeapAlloc for modern code
• INI file functions are deprecated — use registry or XML instead
• Fibers are user-mode scheduling units — complex to use correctly
• Atoms are global string identifiers — limited to 255 characters
• CreateFileTransactedW (TxF) requires Windows Vista+ and is partially deprecated
• These functions exist for legacy application compatibility
